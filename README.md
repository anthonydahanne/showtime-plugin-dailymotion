Dailymtion plugin for Showtime (Unofficial)
============================

Dailymotion.com is a website to share videos.

Showtime (https://www.lonelycoder.com/showtime ) is a great media center for PS#, Linux and MacOSCX

Dailymotion Showtime plugin allows the integration of Dailymotion videos into Showtime.

This plugin is not endorsed by Dailymotion in any way, and is available free of charge

Git: https://github.com/anthonydahanne/showtime-plugin-dailymotion

## Release notes

1.0 - First release

- allows you to search videos using a search criteria, and then select and play a video
- finds the best resolution for oyur content

## Installing

Dailymotion Showtime plugin, is available directly from Showtime plugin repository. Just install it from there (the puzzle peace in showtime homepage right top corner).

Dailymotion Showtime plugin, *needs* Showtime version >= 4.2

- Official versions: https://www.lonelycoder.com/showtime/download
